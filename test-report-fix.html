🔧 ECR报告页面测试脚本

📋 使用说明：
1. 确保开发服务器运行在 http://localhost:5173
2. 打开浏览器开发者工具 (F12)
3. 切换到 Console 标签
4. 复制并执行下面的脚本
1 创建测评并快速完成
// 导入store const { useAppStore } = await import('/src/store.ts') const store = useAppStore() // 创建新测评 console.log('🚀 创建新测评...') const assessmentId = await store.createNewAssessment() console.log('✅ 测评创建成功，ID:', assessmentId) // 快速完成测评（安全型） console.log('⚡ 快速完成测评...') await store.quickCompleteAssessment('secure') console.log('✅ 测评完成！') // 检查数据 console.log('📊 检查测评数据:', store.currentAssessment) console.log('📈 检查结果数据:', store.currentAssessment?.result)
2 跳转到报告页面
// 跳转到基础报告页面 const reportUrl = `/report/${assessmentId}` console.log('🔗 跳转到报告页面:', reportUrl) window.location.href = reportUrl
⚠️ 如果仍然看到"测评尚未完成"错误：
请在控制台中运行调试脚本检查数据状态
3 调试脚本（如果报告页面仍有问题）
// 调试当前状态 console.log('=== 调试信息 ===') console.log('Current URL:', window.location.href) console.log('Assessment ID from URL:', window.location.pathname.split('/')[2]) // 重新获取store const { useAppStore } = await import('/src/store.ts') const store = useAppStore() console.log('Store assessment:', store.currentAssessment) console.log('Has assessment:', !!store.currentAssessment) console.log('Assessment result:', store.currentAssessment?.result) console.log('Assessment basicResult:', store.currentAssessment?.basicResult) // 如果没有数据，尝试加载 const urlAssessmentId = window.location.pathname.split('/')[2] if (urlAssessmentId && (!store.currentAssessment || store.currentAssessment.id !== urlAssessmentId)) { console.log('🔄 尝试加载测评数据...') const success = await store.loadAssessment(urlAssessmentId) console.log('加载结果:', success) console.log('加载后的数据:', store.currentAssessment) }
4 快速测试不同依恋类型
// 测试不同依恋类型 const testTypes = ['secure', 'anxious', 'avoidant', 'disorganized'] for (const type of testTypes) { console.log(`🧪 测试 ${type} 类型...`) // 创建新测评 const id = await store.createNewAssessment() await store.quickCompleteAssessment(type) console.log(`✅ ${type} 类型测评完成`) console.log(`📊 结果:`, store.currentAssessment?.result) // 跳转到报告页面进行验证 console.log(`🔗 访问报告: /report/${id}`) }
✅ 预期结果：
1. 能够成功创建和完成测评
2. 报告页面显示正确的依恋类型和分数
3. 基础报告和详细报告都能正常显示
4. 不再出现"测评尚未完成"的错误
